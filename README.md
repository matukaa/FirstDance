## Features

- Tracks the loading spell with a configurable on-screen icon
- Shows cooldown swipe and countdown text during the loading window
- Plays a configurable TTS alert when First Dance becomes ready
- Supports in-game configuration for icon visibility, position, size, and ready TTS settings
- Opens the configuration panel with `/firstdance` or `/fd`

## Installation

1. Download or clone this repository.
2. Place the `FirstDance` folder in your WoW AddOns directory:
   `World of Warcraft/_retail_/Interface/AddOns/`
3. Reload the game or restart WoW.

## Usage

- Use `/firstdance` or `/fd` to open the configuration panel.
- Enable or disable the loading icon.
- Use `Test Icon` to preview icon placement and size.
- Adjust icon offset and size from the config panel.
- Enable or disable ready TTS and configure its text, rate, and volume.
- Use `Test` to preview the ready TTS line.
